# Rock Paper Scissors Java Script Game

### Repository for DUMP Internship: Week 12 - JS #2

<b>Steps to run application</b>

```
git clone https://github.com/DorianLeci/Internship-12-RPS-GAME.git
cd Internship-12-RPS-GAME
```

You can open index.html by opening it in browser as file or through VS Code extension <b>LiveServer</b>.
</br>
You can find installation info and documentation here: https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
</br>
</br>

<b>App Functionality Info</b>
</br>
Every game has 5 rounds and each time has <b>TIMER</b> which is set to 20 seconds. If 20 seconds pass random move for you is generated.<br/>
Every round is played against bot which has pre-generated moves.</br>
At the end of game you can click on "Game overview" button to see moves and winner for each of the 5 rounds.
